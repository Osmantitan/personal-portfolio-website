# Personal Portfolio Website

A modern, responsive personal portfolio website showcasing my skills, education, and projects as a web developer.

## Live Demo

Visit my portfolio: [https://osmantitan.github.io/personal-portfolio-website/](https://osmantitan.github.io/personal-portfolio-website/)

##  About

This is my personal portfolio website built to showcase my journey as a Diploma student in Information Technology. The website features information about my education, skills, hobbies, and interests.

## features

- Clean and modern design
- Responsive layout (works on mobile, tablet, and desktop)
- Easy navigation between pages
- Downloadable resume
- Contact information

## Built With

- HTML5
- CSS3
- Bootstrap 5


## Pages

- **Home**: Introduction and overview
- **About**: Detailed information about my education, skills, and interests
- **Contact**: Ways to get in touch with me
